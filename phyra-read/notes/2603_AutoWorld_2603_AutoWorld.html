01

Problem & Motivation

Core Question

Can a model harness unlabeled LiDAR data to improve multi-agent traffic simulation performance, bypassing the labeled-trajectory bottleneck?

Supervised Scaling Bottleneck

All existing simulation models (diffusion, autoregressive, token-based) depend on annotated trajectory datasets. Annotation cost directly caps performance scaling.

Semantic Label Dependency

Prior LiDAR world models require semantic labels to distinguish dynamic from static regions. This prevents fully self-supervised world model training.

Uniform Supervision Bias

Without labels, occupancy grids are dominated by static voxels. Uniform loss weighting biases learning away from motion-critical dynamic regions.

Diversity-Realism Trade-off

IID sampling from generative models produces redundant scenarios. Existing diversity methods do not account for per-sample quality, risking off-manifold outputs.

Method Overview

Two-Stage Framework

Unlabeled LiDAR sequences └─> Voxelize -> VAE encode -> Past latents Z_{t-Th:t} └─> Rectified-flow World Model µ_θ (motion-aware weighted loss) └─> N future latent rollouts Z_{t+1:t+Tf} [DPP scene diversity] └─> MHCA pooling -> Global context g (coarse) └─> Per-step condition c^f_t = MHCA([ẑ_t ; g]) (fine) └─> Conditional Diffusion Model f_ψ [DPP motion diversity] └─> N × M diverse, plausible scenarios

Module 1: Motion-Aware Latent Occupancy World Model

Rectified flow model predicting future latent occupancy. Loss is reweighted spatially toward dynamic voxels detected from temporal occupancy change maps.

$$\mathcal{L}^{\text{world}}(\theta) = \mathbb{E}_{k_\mu, z_0, \tilde{z}} \sum_{i,j} W_t(i,j) \left\| \mu_\theta(z_{k_\mu}, k_\mu, c^\mu)_{i,j,:} - (\tilde{z}-z_0)_{i,j,:} \right\|_2^2$$

Weight map: $W_t(i,j) = 1 + \lambda\bar{C}_{t,\Delta}(i,j)$, normalized to unit spatial mean. No semantic labels required.

Module 2: Coarse-to-Fine World-Model Conditioning

  • Coarse (global): $g = \text{MHCA}(Q=\xi, \; K=V=\phi_1(\{\hat{z}_{t+e\delta}\}))$ — learned token aggregates the full latent sequence.
  • Fine (per-step): $c^f_t = \text{MHCA}(Q=h, \; K=V=\phi_2([\hat{z}_t; g]))$ — scene encoding $h$ from query-centric Transformer as queries.

Module 3: Cascaded Latent DPP Diversity (inference-time, no retraining)

  • Quality-weighted kernel: $\kappa_q(x^{(i)}, x^{(j)}) = q^{(i)} \kappa(x^{(i)}, x^{(j)}) q^{(j)}$
  • Level 1 (scene): DPP repulsion over world-model latent rollouts; quality = FVD-based score.
  • Level 2 (motion): DPP repulsion over trajectory embeddings; quality = constraint violations (collision, off-road).
  • Gradient $\nabla \log P^q_\kappa$ added as repulsive shift at each ODE / diffusion reverse step.

Highlights

  • First-rank RMM (0.7865) on WOSAC without RL/adversarial fine-tuning (Table 1, p. 10)
  • MA supervision closes most IoU_dynamic gap to semantic training without labels (Table 3, p. 13)
  • Monotonic RMM scaling from 1x to 5x unlabeled data (Fig. 4, p. 14)
  • Cascaded DPP is training-free; quality weighting prevents off-manifold diversity
  • Scene diversity alone degrades performance without motion diversity (Table 2)

Weaknesses

  • Leaderboard margin of 0.0001 RMM over DecompGAIL; no confidence intervals reported
  • Ablations use only 2% of validation split, reducing statistical reliability
  • World model evaluated only on WOD; cross-domain transfer not examined
  • Code not yet released at submission
  • FVD as DPP quality proxy; its correlation with downstream RMM is implicit only

Structural Limitations

  • Two-stage training leaves the world model blind to downstream RMM; coupling is indirect (conditioning only, no joint signal).
  • DPP diversity operates on latent geometry that may not preserve semantic behavioral diversity.
  • LiDAR dependency: the approach does not extend to camera-only setups without re-engineering the occupancy representation.
  • DPP kernel matrices are O(N²) and O(M²); quadratic cost caps practical sample budgets.

Red Flags

Leaderboard comparison (Table 1, p. 10): top-6 methods span only 0.0014 RMM. No statistical significance testing is reported. The 0.0001 first-rank margin may not be reproducible across different evaluation seeds.
Ablation scope (Table 2, p. 12): 2% validation split is used to contain evaluation cost; per-cell absolute differences are small enough that the ranking could reverse with different random samples.
