Problem: Fragmented 4D Driving Scene Generation

Existing methods for generating dynamic driving scenes face a three-way trade-off. No prior work achieves all four desirable properties simultaneously:

Key insight: Metric depth serves as a geometric bridge between temporal generation and spatial NVS, enabling disentangled yet coherent 4D scene synthesis.

Method Overview: Disentangled Two-Stage Framework

Historical RGB + Control Signals (Traj, BEV, BBox, CamPose) | v [DiST-T: Diffusion Transformer] ---> Future Multi-Camera RGB-D Sequences | | | Aggregate to | Point Clouds v | Project to Novel Viewpoint <-------------------+ | v Sparse RGB-D Conditions | v [DiST-S: Denoising UNet + Condition Encoder] | v Dense Novel View RGB-D Output

The framework decomposes 4D generation into: (1) DiST-T for temporal RGB-D prediction, and (2) DiST-S for spatial novel view synthesis using projected RGB-D as sparse conditions.

Metric Depth Curation Pipeline

A pseudo-GT curation pipeline produces dense metric depth maps for training:

  1. Multi-view stereo (MVS) generates static scene point clouds from multi-camera images
  2. LiDAR aggregation (3-frame window) fused with MVS; LiDAR prioritized where available
  3. Semantic filtering removes dynamic objects and sky via 2D segmentation
  4. Generative depth completion refines the merged depth prompt into dense metric depth maps

Depth range: 0-100m; sky assigned 100m; voxel resolution 0.1m.

DiST-T: Temporal RGB-D Generation

A diffusion transformer that jointly predicts future multi-camera RGB and depth sequences.

  • Backbone: STDiT (N=28 blocks, d=1152), with temporal and spatial attention blocks
  • VAE: Pre-trained 3D VAE from CogVideoX; RGB and depth encoded separately
  • Control injection: ControlNet-style branch encodes map M, reference images, 3D bounding boxes B, trajectory A, and camera poses P
  • Training: Progressive resolution (224x400 then 424x800), rectified flow + v-prediction loss, on 8x NVIDIA H20 GPUs
$$\{\mathbf{Z}_t^I, \mathbf{Z}_t^D\} = \mathcal{G}_\theta(\{\mathbf{S}_t, \mathbf{I}_{\text{ref}}\})$$

DiST-S: Spatial Novel View Synthesis

A denoising UNet that synthesizes dense RGB-D at novel viewpoints from sparse projected conditions.

  • Base model: SVD (Stable Video Diffusion), resolution 448x768
  • Input conditions: Projected sparse RGB-D + valid mask from point cloud reprojection
  • Condition encoder: Trainable encoder for sparse inputs, concatenated with frozen VAE features
  • Input channels: 8 to 16 (RGB + depth); output channels: 4 to 8
$$\{\mathbf{Z}_{\text{tgt}}^I, \mathbf{Z}_{\text{tgt}}^D\} = \mathcal{F}_\theta(\{\mathbf{Z}_{\text{cond}}^I, \mathbf{Z}_{\text{cond}}^D\})$$

Self-Supervised Cycle Consistency (SCC)

A two-stage training strategy that enables DiST-S to generalize to unseen viewpoints:

  1. Stage 1: Train DiST-S on original trajectories from training set
  2. Stage 2: Generate novel trajectories by lateral shift $\tau \in [-3m, +3m]$; synthesize novel views; reproject back to original viewpoints; train DiST-S on cycle-consistent pairs

Result: FID at +/-2m shift drops from 15.72 to 14.97 (DiST-4D), and from 13.00 to 10.12 (DiST-S standalone), approximately 20% improvement (Table 5, p.7).

Highlights

  • First unified feed-forward pipeline for 4D driving scene generation (Table 1)
  • High-quality metric depth curation pipeline (Abs. Rel. 0.13, Table 7)
  • SCC strategy reduces FID by 20% for novel views

Limitations

  • Pseudo GT quality degrades for distant objects and sky
  • Reference depth shows side effects on RGB forecasting (Table 2)
  • Single-dataset evaluation (nuScenes only)

Critical Assessment

Contribution Validation

  • Supported: First feed-forward 4D generation framework; confirmed by Table 1 feature comparison and competitive quantitative results
  • Partially supported: Metric depth as "seamless bridge"; Table 2 shows Ours-D has side effects on RGB quality, suggesting imperfect integration
  • Supported: Depth curation quality exceeds baselines (Table 7: 0.13 vs 0.27 Abs. Rel.)
  • Supported: SCC enhances generalization (Table 5: consistent improvements across all shifts)

Structural Limitations

  • Sequential dependency: DiST-T errors in depth cascade to DiST-S without correction feedback
  • Limited viewpoint range: quality degrades rapidly beyond +/-3m training range (FID 17.57 at +/-4m vs 10.12 at +/-1m)
  • Pseudo GT ceiling: distant object depth quality bounds the entire framework's spatial accuracy
