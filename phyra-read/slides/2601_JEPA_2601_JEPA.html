Problem and Motivation

Context

JEPA world models learn predictive representations via self-supervised objectives and use MPC for action planning. However, the MPC cost landscape in representation space contains numerous local minima, degrading planning quality.

Core Issue

Representations optimized for prediction accuracy are not necessarily well-suited for planning. The Euclidean distance between embeddings does not reflect the actual goal-reaching cost, and is symmetric while reaching costs are generally asymmetric.

This Paper's Proposal

Shape the JEPA representation space using an IQL loss so that the Euclidean distance (or quasi-distance) between state embeddings approximates the negative goal-conditioned value function. This aligns the MPC optimization landscape with the true planning objective.

Methodology

Technical Approach

Train the JEPA state encoder with an IQL-inspired loss: enforce that $V_\theta(s,g) = -\|E_\theta(s) - E_\theta(g)\|_2$ approximates the goal-conditioned value function $V^*$. At planning time, MPC/MPPI minimizes this value function over action sequences.

Data Flow

Observation s ──▶ State Encoder E_θ ──▶ Embedding z_s │ z_s, z_g ──▶ V_θ(s,g) = -‖z_s - z_g‖ │ Trajectory data ──▶ IQL Loss L_VF │ z_s + action a ──▶ Predictor ──▶ Prediction Loss L_pred │ Total Loss ──▶ Train Encoder + Predictor │ Planning: MPPI minimizes V_θ over action sequences

Key Formula (Eq. 1, p.2)

$$\mathcal{L}_{VF}^{\theta} = \sum_{n=0}^{N} \sum_{t=0}^{T-1} L_2^{\tau}\!\left(-\mathbf{1}_{s_t \neq g_n} + \gamma V_{\theta}(s_{t+1}, g_n) - V_{\theta}(s_t, g_n)\right)$$

where $L_2^{\tau}(x) = |\tau - \mathbf{1}_{x < 0}|\, x^2$ is the expectile regression loss, with discount $\gamma$ and expectile $\tau$.

Quasi-Distance Extension

The reaching cost from state A to B is generally not equal to B to A. Euclidean distance is symmetric and cannot capture this.

The paper replaces Euclidean distance with a quasi-distance (Wang and Isola, 2022), an asymmetric distance measure that better models directional reaching costs.

Two Training Strategies

StrategyDescription
Sep (Separate)Train state encoder with $\mathcal{L}_{VF}$ alone, then train predictor with $\mathcal{L}_{pred}$
JointTrain all networks together with $\mathcal{L}_{VF} + \mathcal{L}_{pred}$

Key Findings

  • VF_quasi achieves the best planning accuracy across all three environments (0.71, 0.96, 0.63), confirming that value-shaped representations with asymmetric distance improve MPC planning (Table 2, p.4)
  • Adding prediction loss to VF methods degrades performance: VF_pred < VF and VF_quasi_pred < VF_quasi across all settings, suggesting the two objectives may conflict
  • VCReg regularization hurts VF methods: VF_VCReg consistently performs worse than VF alone, indicating that diversity-promoting regularization interferes with value function learning
  • Quasi-distance consistently outperforms Euclidean: the asymmetric formulation better captures the directional structure of reaching costs

Hyperparameter Sensitivity (Figure 3, p.7)

Two key hyperparameters control the IQL loss behavior:

  • Discount factor $\gamma$: increasing $\gamma$ improves performance by capturing distant state relationships. Optimal values: $\gamma = 0.98$ for VF, $\gamma = 0.93$ for VF_quasi
  • Expectile $\tau$: should be set close to 1.0, but values too close to 1 cause training instabilities. Optimal: $\tau = 0.80$ for VF, $\tau = 0.60$ for VF_quasi

The separate representation approach (training two encoder spaces, one for prediction and one for value) did not improve results (planning accuracy 0.60 on WS, p.7).

Limitations and Discussion

Locality Problem (p.4)

Value function gradients become small for states far from the goal, producing low signal-to-noise ratios. Distant state relationships are poorly captured. The authors suggest hierarchical representation spaces as a potential remedy.

Dataset Dependence (p.4)

IQL requires adequate state-space coverage in training trajectories. The method is biased in random environments, and training states must span the entire state space for the IQL loss to be effective.

Experimental Scope

Only two toy environments tested (64x64 grid worlds). No evidence of scalability to complex visual domains, continuous control benchmarks, or high-dimensional action spaces.

Phyra's Judgment

Supported Claims

The claim that value-function-shaped representations improve JEPA planning is supported: VF_quasi achieves the best results across all settings. The systematic comparison of 10 training approaches provides clear evidence for the value of IQL-based representation shaping.

Overclaimed Aspects

The abstract's claim of "significantly improved planning performance" lacks statistical testing. The claim about "mitigating local minima" (p.1) is plausible but unverified; no loss landscape analysis is provided.

Fundamental Constraints

  • Training data coverage is a hard requirement that cannot be relaxed within the current framework
  • The locality problem is structural to the distance-based value function formulation
  • Scalability beyond toy environments remains entirely undemonstrated
