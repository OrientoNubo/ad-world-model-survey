Generating Multimodal Driving Scenes via Next-Scene Prediction
Autonomous Driving Multimodal Generation Autoregressive World Model Traffic Simulation

Research Overview

UMGen is an autoregressive framework that generates ego-centric driving scene sequences covering four modalities simultaneously: ego-vehicle action, traffic map, traffic agents, and camera image. It decomposes generation into inter-frame temporal prediction (TAR) and intra-frame ordered prediction (OAR), with an Action-aware Map Alignment (AMA) module maintaining spatial coherence between predicted ego-action and map features across frames. The system supports fully autonomous generation from an initial scene, as well as user-controlled ego-action and agent-action injection for closed-loop scenario authoring.

Core Pain Points

  • Static map: GUMP/TrafficGen fix the map segment; real ego-centric view shifts dynamically as the vehicle moves.
  • No agent motion in video models: DriveDreamer/GAIA-1 produce images but cannot control per-agent trajectories.
  • Vanilla AR complexity: Concatenating all T-frame tokens yields O((TN)^2) attention, prohibiting long-horizon generation.
  • Intra-frame inconsistency: Temporal-only AR decoding allows cross-modal token conflicts within the same scene frame.

TAR — Temporal AutoRegressive Module

Applies causal self-attention position-wise across T historical frames for each of the N token positions independently, then a bidirectional self-attention block integrates across positions within the predicted frame. Complexity: O(T²) vs. O((TN)²) for vanilla AR.

$$\tilde{e}^i_{T+1} = \text{CSA}(\tilde{e}^i_1, \ldots, \tilde{e}^i_T) \qquad h_{T+1} = \text{SA}(\tilde{e}^1_{T+1}, \ldots, \tilde{e}^N_{T+1})$$

OAR — Ordered AutoRegressive Module

Decodes tokens in fixed modality order (ego-action → map → agent → image) using TAR features $h_{T+1}$ as temporal priors injected at every decoding step. Prevents intra-frame cross-modal conflicts.

$$w_j = o^j_{T+1} + h^{j+1}_{T+1}, \quad o^i_{T+1} = \text{CSA}(w_1,\ldots,w_{i-1})$$

AMA — Action-aware Map Alignment Module

Reshapes map tokens to a spatial grid $F_T$, then applies an affine transformation parameterised by the predicted ego-action $(θ, dx, dy)$ to produce an aligned prior map for the next frame.

$$\begin{bmatrix}x'\\y'\end{bmatrix} = \begin{bmatrix}\cos\theta & -\sin\theta\\\sin\theta & \cos\theta\end{bmatrix}\begin{bmatrix}x\\y\end{bmatrix} + \begin{bmatrix}dx\\dy\end{bmatrix} \qquad \tilde{e}^{map}_T = \text{Flatten}(\tilde{F}_T) + e^{map}_T$$

Highlights

First framework to jointly generate all four primary AD modalities including a dynamic ego-centric map, filling a gap left by two-modality predecessors (Table 1).
TAR reduces inter-frame attention from O((TN)²) to O(T²), enabling generation up to 60-second sequences (Fig. 6 validates stable GPU memory vs. growing vanilla AR).
OAR reduces agent collision rate from 5.6% to 5.1% (Table 4), a measurable intra-frame consistency gain.

Red Flags

No quantitative sequence metric. Sequence realism is demonstrated qualitatively only; no FID or distributional metric covers multi-frame temporal coherence, making cross-method numerical comparison impossible.
AMA ablation is qualitative-only. Figure 7 shows one example; no map MMD or realism score is reported with/without AMA, making magnitude assessment impossible.
Heterogeneous baselines for initial scene MMD. Compared methods do not generate images or maps; the four-modality consistency claim is not directly validated against any competitor.

Structural Limitations

  • TAR position-wise attention skips cross-position temporal correlations; deferred to a single-frame bidirectional SA, which may underfit complex spatial-temporal patterns.
  • AMA affine transform is an approximation; fails at intersections, elevation changes, and occlusion boundary shifts not captured by a 2D rigid body motion.
  • Fixed intra-frame token order encodes a hard causal assumption that may prevent beneficial feedback loops (e.g., image features informing agent prediction).
  • No analysis of distribution shift when user-injected actions place the vehicle in out-of-distribution states.

Open Questions and Improvement Directions

Outstanding Questions

  • How does quality degrade beyond the 20-frame training block size? The 60-second claim requires an explanation of the generalization mechanism.
  • How sensitive is AMA to ego-action prediction errors? Error propagation through the affine transform is not analyzed.
  • Does the 1028-entry agent codebook provide sufficient attribute resolution for fine-grained agent behaviors?
  • How does the system behave on out-of-distribution user inputs (extreme speeds, off-road scenarios)?

Improvement Directions

  • (High feasibility) Adopt FVD or closed-loop collision rate as a quantitative sequence metric to replace qualitative-only assessment.
  • (High feasibility) Report map MMD and realism scores from Appendix C with/without AMA to replace the qualitative-only ablation.
  • (Medium feasibility) Use masked autoregressive decoding instead of fixed modality order to allow bidirectional cross-modal conditioning.
  • (Medium feasibility) Promote the diffusion image decoder to primary output (FID drops ~6 points; Tables 8 in appendix show consistent gain across all sequence lengths).
