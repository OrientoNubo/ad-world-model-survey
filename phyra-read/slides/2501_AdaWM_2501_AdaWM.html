AdaWM: Adaptive World Model based Planning for Autonomous Driving

Hang Wang, Xin Ye, Feng Tao, Chenbin Pan, Abhirup Mallik, Burhaneddin Yaman, Liu Ren, Junshan Zhang
World Model Autonomous Driving Reinforcement Learning Online Finetuning

Problem and Motivation

World model based RL uses a pretrain-finetune paradigm for autonomous driving. However, naive finetuning causes performance degradation due to distribution shift between pretraining and new tasks.

  • Existing methods either lack finetuning support (VAD, UniAD) or use single-focus strategies (model-only or policy-only)
  • Alternating model+policy updates does not distinguish which component causes more degradation
  • The dominant source of mismatch varies across tasks and across finetuning steps

Core Insight: Two Sources of Degradation

Dynamics Model Mismatch

The pretrained dynamics model fails to predict the new task's state transitions accurately, measured by TV distance $D_{TV}(P \| \hat{P})$ between true and predicted state-action visitation distributions.

Policy Mismatch

The pretrained policy becomes suboptimal for the new environment, measured by TV distance $D_{TV}(\pi_t \| \pi_\omega)$ between current and pretrained action distributions.

$$\eta - \hat{\eta} \leq \underbrace{\gamma^K \frac{\mathcal{E}_{\max}}{1-\gamma^K} + \Gamma\frac{2\gamma\mathcal{E}_P}{1-\gamma^K}}_{\text{dynamics mismatch}} + \underbrace{\Gamma\left(\frac{4r_{\max}\mathcal{E}_\pi}{1-\gamma} + \frac{4\gamma\mathcal{E}_\pi}{1-\gamma^K}\right)}_{\text{policy mismatch}}$$

Theorem 1 (p.6): Performance gap upper bound decomposed into two mismatch terms.

AdaWM: Method Overview

At each finetuning step, AdaWM performs two phases:

  • Mismatch Identification: Estimate TV distances for dynamics model and policy to determine which mismatch dominates
  • Alignment-driven Finetuning: Selectively update the dominant component using efficient low-rank methods

Efficient Finetuning Mechanisms

Dynamics Model: LoRA

Parameters decomposed into base vector $Z$ and vector $\Phi$. Only the weight $B$ of the base vector is updated during finetuning:

$B' \leftarrow B, \quad \phi' = (B'Z)^\top\Phi$

Based on NoLA (Koohpayegani et al., 2024)

Policy: Weighted Convex Ensemble

Policy network decomposed into sub-units: $\Omega = \sum_{i=1}^{D}\delta_i\omega_i$. Only the weight vector $\Delta$ is updated:

$\Delta' \leftarrow \Delta$

Reduces trainable parameters while preserving structure

Evaluation and Limitations

Strengths

  • Principled theoretical foundation (Theorem 1) for the adaptive decision
  • Consistent improvement over all baselines on all 4 tasks
  • Clear motivating example demonstrating the problem

Limitations

  • CARLA-only validation; no real-world or multi-simulator experiments
  • Fixed threshold C requires task-specific tuning
  • Binary per-step decision (model xor policy); no soft allocation
