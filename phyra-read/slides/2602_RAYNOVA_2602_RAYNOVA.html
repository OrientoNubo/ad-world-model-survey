World Model Autoregressive Multi-view Video Autonomous Driving Ray Space Plucker Coordinates

Problem and Motivation

World foundation models aim to generate physically plausible multi-view videos for autonomous driving. Prior methods treat spatial and temporal correlations separately, leading to two categories of limitations:

  • Decoupled spatial-temporal design (e.g., temporal VAEs, cross-view projections): assumes fixed camera configurations, limiting generalization to novel sensor setups (p.1)
  • Explicit 3D priors (e.g., point clouds, BEV features): binds the model to specific coordinate systems, restricting adaptability to heterogeneous training data (p.1)

RAYNOVA's core argument: by encoding visual tokens through their relative positions in a continuous 7D Plucker ray space, the model can unify spatial and temporal reasoning without hard-coded geometric assumptions.

Method: Dual-Causal Autoregression

RAYNOVA is built on a "next-scale prediction" visual autoregressive model (Infinity [19]) and extends it with two folds of causality:

  • Scale causality: Multi-scale token maps $X_1, \ldots, X_K$ are generated coarse-to-fine, where each scale conditions on all previous scales (Eq. 1, p.4)
  • Time causality: Multi-view images at each timestep condition on all views from all past frames, without assuming inter-camera dependencies within the same timestep (Eq. 3, p.5)
$$p\left(X_1^{1:V,1:T}, \ldots, X_K^{1:V,1:T}\right) = \prod_{t=1}^{T} \prod_{k=1}^{K} p\left(X_k^{1:V,t} \mid X_{1:k-1}^{1:V,t}, X_{1:K}^{1:V,1:t-1}\right)$$

Eq. 4 (p.5): The dual-causal block formulation, unifying scale and time autoregression.

Isotropic Spatio-Temporal Representation

Each visual token is associated with a Plucker ray $\mathbf{p}_k^{v,t} = (\mathbf{m}_k^{v,t} \in \mathbb{R}^3, \mathbf{d}_k^{v,t} \in \mathbb{R}^3, t)$ in 7D space (Eq. 5, p.5).

  • Relative embedding: Attention scores are modulated by the relative position between camera rays using Rotary Position Embedding (RoPE) extended to 7D (Eq. 6-8, p.6)
  • Isotropic: This representation is the same for all scales, views, and frames, enabling the model to learn interactions without spatial or temporal inductive biases

Ablation (Tab. 6, p.9): Relative embedding achieves FID 17.2 / FVD 124, vs. Absolute (17.2 / 138) and None (18.7 / 214), confirming the value of relative ray-level encoding for temporal coherence.

Dual-Causal Block Architecture

Each transformer block contains three attention modules executed sequentially (Sec. 3.4, p.6):

  1. Image-wise self-attention: Processes each image independently for per-image realism; inherits pretrained weights from Infinity [19]
  2. Global causal attention: Unified self-attention over all prefix tokens across scales, views, and frames; uses relative ray-level RoPE for spatio-temporal consistency
  3. Image-wise cross-attention: Aligns visual tokens with condition inputs (text, 3D bounding boxes, HD maps); randomly initialized with zero-initialization

Ablation (Tab. 8, Appendix B, p.13): Global attention (FID 10.5 / FVD 91) outperforms decoupled per-camera + cross-frame design (FID 15.6 / FVD 140).

Recurrent Training for Long-Horizon Videos

Autoregressive generation over sliding windows causes distribution drift in long sequences. RAYNOVA introduces a recurrent training paradigm (Alg. 1, p.7):

  • Forward and backward propagation are performed frame by frame; gradients accumulate until the end of each sequence
  • KV projection layer latents (not full KV caches) are stored to save 50% GPU memory (p.7)
  • Random bit-wise errors are injected into input tokens during training to simulate inference-time prediction errors (Tab. 9, p.13: FVD improves from 142 to 124 with this technique)

Red Flags and Limitations

  • Domain limitation: All training and evaluation data come from autonomous driving (nuScenes + nuPlan). The "versatile world foundation model" framing overstates the demonstrated scope (Sec. 5, p.10; Appendix D, p.14).
  • Missing in-house comparisons: The authors explicitly note they do not include in-house data baselines (Sec. 4.2, p.8), which limits the reliability of FID/FVD comparisons since training data scale differs.
  • Map condition fidelity: Projection of bounding boxes to camera space is "not perfect due to the lack of height information" (p.8), indicating a structural limitation in the condition encoding design.
  • Error accumulation: Despite the recurrent training mitigation, long-horizon generation still relies on feeding predicted tokens as input, and the paper does not quantify how quality degrades beyond 20s of generation.

Key Takeaways

  1. Relative position encoding in Plucker ray space enables camera-configuration-agnostic world modeling, a meaningful step toward generalizable simulation.
  2. Dual-causal autoregression (scale + time) provides a principled way to model multi-view, multi-frame dependencies without decoupling spatial and temporal dimensions.
  3. The recurrent training paradigm with random error injection effectively reduces distribution drift in long-horizon generation, with FVD improving from 100 to 91 (p.9).
